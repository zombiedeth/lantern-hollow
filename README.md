# Lantern Hollow 🌱✨

A cozy top-down dusk garden game built in **Godot 4.7**. Plant glowing seeds, tend them, watch them bloom, harvest glow-currency, unlock upgrades (lanterns, moonwells, constellations, fairies, ascension), and grow a permanent stardust-powered empire across runs.

## Play it now — in your phone browser

The pre-built web export lives in **[`build/web/`](build/web/)** — drop it on any static host and open `index.html`:

- **Local quick-test:** `cd build/web && python3 -m http.server 8000` then open `http://<your-mac-ip>:8000/` on your phone
- **Static host:** point Netlify / Vercel / GitHub Pages at the `build/web/` folder — `index.html` is the entry point

## Run from source

Requires Godot 4.7+.

1. Install [Godot 4.7](https://godotengine.org/download/)
2. Open the project folder
3. Press **F5** to run
